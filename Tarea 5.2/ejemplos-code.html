Tiene la particularidad de generar un texto con un formato diferente

Con la etiqueta code puedo generar un texto con una apariencia diferente lo cual me viene bien para escribir codigo fuente de algun lenguaje de programación. Esto seria un ejemplo de texto usando la etiqueta code

  • Se utiliza para indicar al navegador que el contenido de dicha etiqueta es un bloque de código fuente de alguno de los lenguajes de programación o marcado
  • Es una etiqueta de línea, se renderiza con la fuente predeterminada monospace que tenga el navegador
  • Atributos comunes y eventos
  • Lo normal será darle estilos css asociados al tipo de lenguajes
